Getting Started with SQL Notebook

This notebook is an SQLite database. Use the CREATE menu to begin adding new data and queries to the notebook. Existing files and databases can be accessed using the IMPORT menu.

For a complete description of SQL Notebook's capabilities and usage, press F1 to open the built-in documentation.
