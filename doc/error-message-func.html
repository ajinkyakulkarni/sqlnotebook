ERROR_MESSAGE Function

Returns the error message set by the last THROW statement.

Syntax

ERROR_MESSAGE()

Return Value

The last error message. This value may be of any type (not just text).

Examples

  • BEGIN TRY  
    THROW 123, 'Message', 0;
END TRY  
BEGIN CATCH  
    PRINT ERROR_MESSAGE();
END CATCH;
    Prints "Message".
