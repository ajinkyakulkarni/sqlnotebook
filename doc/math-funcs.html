Algebra and Trigonometry Functions

A variety of commonly used math functions are available. They all operate on FLOAT values.

Syntax

ACOS(x)
ATAN(x)
ATAN2(x, y)
CEILING(x)
COS(x)
COSH(x)
EXP(x)
FLOOR(x)
LOG(x)
LOG10(x)
POW(x)
ROUND(x)
SIN(x)
SINH(x)
SQRT(x)
TAN(x)
TANH(x)

Examples

  • PRINT POW(2, 5);
    Prints 32.
  • PRINT SIN(3.14 / 2);
    Prints 0.999999682931835.
  • PRINT CEILING(-1.6);
    Prints -1.
  • PRINT ROUND(5.5);
    Prints 6.
