Error Functions

SQL Notebook provides structured error handling using TRY/CATCH statements. Inside the CATCH block, the following functions may be used to retrieve the error information provided by the THROW statement.

  • ERROR_NUMBER() Function
  • ERROR_MESSAGE() Function
  • ERROR_STATE() Function

ERROR_NUMBER() Function

Syntax: SELECT ERROR_NUMBER()

Returns the error number (the first value passed to THROW). If no error number was provided by the THROW statement, then null is returned. The value may be of any type; the THROW statement does not restrict the types of its arguments.


ERROR_MESSAGE() Function

Syntax: SELECT ERROR_MESSAGE()

Returns the error message (the second value passed to THROW). If no error message was provided by the THROW statement, then null is returned. The value may be of any type; the THROW statement does not restrict the types of its arguments.


ERROR_STATE() Function

Syntax: SELECT ERROR_STATE()

Returns the error state (the third value passed to THROW). If no error number was provided by the THROW statement, then null is returned. The value may be of any type; the THROW statement does not restrict the types of its arguments.
