ERROR_NUMBER Function

Returns the error number set by the last THROW statement.

Syntax

ERROR_NUMBER()

Return Value

The last error number. This value may be of any type (not just numeric).

Examples

  • BEGIN TRY  
    THROW 123, 'Message', 0;
END TRY  
BEGIN CATCH  
    PRINT ERROR_NUMBER();
END CATCH;
    Prints 123.
