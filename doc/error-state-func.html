ERROR_STATE Function

Returns the error state set by the last THROW statement.

Syntax

ERROR_STATE()

Return Value

The last error state. This value may be of any type (not just numeric).

Examples

  • BEGIN TRY  
        THROW 123, 'Message', 0;
    END TRY  
    BEGIN CATCH  
        PRINT ERROR_STATE();
    END CATCH;
    Prints 0.
