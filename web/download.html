Download & Install

The current version is 0.6.0 and was released on August 14, 2016. (Release history)

This software is open source (MIT license) and available free of charge.

Download SQL Notebook

Required: 64-bit computer running Windows 7 (or newer) with .NET Framework 4.6.

Alternative installation methods:

  • Download Portable .ZIP
    This portable version does not require installation. Extract the .zip to a folder of your choosing and run SqlNotebook.exe to begin using SQL Notebook.

  • Install using Chocolatey
    Run choco install sqlnotebook to install SQL Notebook.
